How to read this

Scoring, tiers & exclusions
Fit score, audience tier, sources & the LinkedIn exclusion ›

Fit = 0.5 vertical + decision-maker relevance + 0.3 payments/outreach access + 0.2 audience. A fit of 0 means the venue is gated as off-topic (player/consumer-facing or off-vertical).

Audience tier (T1/T2/T3) is a reputation / cross-listing tier, NOT real traffic — exact traffic would need a paid API this project doesn't use.

Source: manual-seed human-verified investigation · llm Claude-judged · fallback provisional keyword guess (upgraded to llm once Anthropic credits are back).

LinkedIn communities are excluded and shown here as manual-only — LinkedIn's ToS/API make community discovery non-automatable, so it is never scraped or faked.
