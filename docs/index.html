Five dashboards I built to answer five real growth questions for Speed Wallet, a Bitcoin + stablecoin payments app: which creators to partner with, what creative is working, where to expand, what's trending now, and which merchants to reach. Each is live, rebuilt from real data, and honest about what it can and can't measure. Pick one below.

Creator Dashboard

Partner scouting

Every discovered creator, scored 0–100 as a potential Speed partner across four real dimensions — audience fit, engagement quality, reach, and sponsorship history.

One rule, everywhere

Built the same way

Every dashboard follows one principle: real signal or an honest gap — never a mocked number to fill a chart. If a metric isn't trustworthy yet, the dashboard says so.

Ship what's real, flag what isn't, and keep the reasoning legible.
